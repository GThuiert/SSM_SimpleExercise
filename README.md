# SSM_SimpleExercise
* 这个项目是尚硅谷的一个简单的SSM的CRUD的练习项目，业务流程容易，代码也简单，纯手打。

## 所用技术点
* 基础框架-ssm（SpringMVC+Spring+MyBatis）
* 数据库-MySQL
* 前端框架-bootstrap快速搭建简洁美观的界面
* 项目的依赖管理-Maven
* 分页-pagehelper
* 逆向工程-MyBatis Generator
